## 当前阶段：初始化

## 学习路径

基于学员画像（AI 新手 + Web3 新手 + 无代码），推荐学习路径：

### Phase 1：AI 基础入门（预计 2-3 周）

按侧边栏顺序：

1. [大语言模型（LLM）](https://aiweb3.school/zh/handbook/ai/llm/) — 理解大模型能做什么、不能替代什么
2. [提示词（Prompt）](https://aiweb3.school/zh/handbook/ai/prompt/) — 学会写任务目标、边界和输出格式
3. [上下文（Context）](https://aiweb3.school/zh/handbook/ai/context/) — 模型一次能看到什么
4. [检索增强生成（RAG）](https://aiweb3.school/zh/handbook/ai/rag/) — 外部知识怎么接入模型
5. [智能体（Agent）](https://aiweb3.school/zh/handbook/ai/agent/) — 模型如何调用工具和多步执行

### Phase 2：Web3 基础入门（预计 2-3 周）

1. [网络（Network）](https://aiweb3.school/zh/handbook/web3/network/) — 区块、共识、L2、RPC
2. [密码学（Cryptography）](https://aiweb3.school/zh/handbook/web3/cryptography/) — 哈希、公私钥、签名
3. [钱包（Wallet）](https://aiweb3.school/zh/handbook/web3/wallet/) — 钱包是身份+签名入口
4. [智能合约（Smart Contract）](https://aiweb3.school/zh/handbook/web3/smart-contract/) — 链上规则
5. [账户抽象（Account Abstraction）](https://aiweb3.school/zh/handbook/web3/account-abstraction/) — Smart Account

### Phase 3：AI × Web3 Bridge（预计 2-3 周）

1. [Chain-aware Context](https://aiweb3.school/zh/handbook/bridge/chain-aware-context/)
2. [Web3 Tool Use](https://aiweb3.school/zh/handbook/bridge/web3-tool-use/)
3. [Agent Workflow](https://aiweb3.school/zh/handbook/bridge/agent-workflow/)
4. [Agent Wallet](https://aiweb3.school/zh/handbook/bridge/agent-wallet/)
5. [Machine Payment](https://aiweb3.school/zh/handbook/bridge/machine-payment/)

### Phase 4：Hackathon 项目实践

从 [前沿探索](https://aiweb3.school/zh/handbook/tracks/open-track/) 中选一个方向做原型。
