## 关键收获

- LLM 的核心判断：「模型输出是候选结果，不是事实本身」——对内容运营方向尤其重要
- Agent 的本质：模型 + 工具调用 + 规划 + 状态 + 反思，不是「全自动 AI 员工」
- 做可交互学习产物是「无代码」路线的最佳实践——概念卡片 + 测验不需要写代码，但需要理解概念
- Agent 生成的内容需要人工审阅，尤其是测验题目的措辞和答案准确性
