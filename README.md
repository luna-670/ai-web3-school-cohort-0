# AI × Web3 School — 个人学习仓库

> Personal learning journal and proof-of-work for AI × Web3 School.

## 关于

[AI × Web3 School](https://aiweb3.school) 是由 [LXDAO](https://lxdao.io/) 与 [ETHPanda](https://ethpanda.org/) 共同发起的开源学习计划，面向 builders 系统学习 AI、区块链、智能合约、AI Agent 与去中心化 AI。

本仓库是我在 AI × Web3 School 的个人学习记录，包含每日打卡、任务笔记、实验项目、Handbook 反馈和 Hackathon 项目。
